# Getting Started (v1)

Welcome to the v1 documentation! This guide will help you get started.

## Introduction

This documentation is powered by [VitePress](https://vitepress.dev/) and served through [Laravel](https://laravel.com/).

## Prerequisites

Before you begin, make sure you have:

- PHP 8.0 or higher
- Laravel 9.x or higher
- Node.js 16.x or higher (for building documentation)

## Quick Start

1. Install the package via Composer
2. Run the installation command
3. Visit `/docs` in your browser
