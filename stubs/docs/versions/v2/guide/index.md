# Getting Started (v2)

Welcome to the v2 documentation! This guide will help you get started.

## Introduction

This documentation is powered by [VitePress](https://vitepress.dev/) and served through [Laravel](https://laravel.com/).

## Prerequisites

Before you begin, make sure you have:

- PHP 8.1 or higher
- Laravel 10.x or 11.x
- Node.js 18.x or higher (for building documentation)

## Quick Start

1. Install the package via Composer
2. Run the installation command
3. Visit `/docs` in your browser

## What's New in v2

- Role-based access control
- Permission-based access control
- Custom domain support
- Improved caching
